Citing QRMI
===========

QRMI is the work of `many people`_ who contribute to the project at different levels.
If you use QRMI, please consider citing the associated overview paper,
`"Quantum resources in resource management systems"`_. This helps support the continued
development and visibility of the repository. The BibTeX citation handle can be found in the `BibTeX file`_.

.. _many people: https://github.com/qiskit-community/qrmi/graphs/contributors
.. _"Quantum resources in resource management systems": https://arxiv.org/abs/2506.10052
.. _BibTeX file: https://github.com/qiskit-community/qrmi/blob/main/CITATION.bib

.. note::

    We expect multiple versions of the overview paper to be released as the project evolves.
